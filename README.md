# SDAE_IPIAL
**SDAE IPIAL (Sistema De Acesso A Exames no IPIAL)**
É um sistema de gestão escolar, destinado para a gestão da inscrição de dos novos alunos na escola, o sistema tem a finalidade de otimizar, automatizar e facilitar esse processo, evitando atrasos e outros erros que podem prejudicar o bom funcionamento da instituição.

O mesmo foi feito no âmbito de uma avaliação prática realizada por mim (António Garcia) e por meus colegas na disciplina de TLP (Tecnica de Linguagem de Programação).

Este repositório tem como finalidade:

* Quardar os arquivos do projecto;

* Fazer a continuação e (alteração e eatualização) do projeto.
