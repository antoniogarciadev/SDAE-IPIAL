# SDAE-IPIAL
SDAE IPIAL (Sistema De Acesso A Exames no IPIAL) É um sistema de gestão escolar, destinado para a gestão da inscrição de dos novos alunos na escola, o sistema tem a finalidade de otimizar, automatizar e facilitar esse processo, evitando atrasos e outros erros que podem prejudicar o bom funcionamento da instituição.
